## 四、马哲高频考点（快速识别题型）

| 题目特征 | 答案方向 |
|---------|---------|
| "既...又..." / "分析形势既...也..." | **两点论** |
| 一个国家走向现代化既遵循一般规律，更符合本国实际 | **矛盾普遍性与特殊性** |
| 手推磨→封建主义，蒸汽磨→资本主义 | **生产力决定生产关系** |

| 泾溪石险人兢慎，却是平流无石处时时闻说有沉沦 | **矛盾双方可以转化（居安思危）** |

## 五、经济常识快查

| 考点 | 答案 |
|------|------|
| 社会总供给＜总需求，应采取 | **宽松的货币政策** |
| 财政政策+货币政策 = 宏观调控两大手段 | 财政政策（税收/支出）+ 货币政策（利率/准备金率） |
| 基尼系数：越小代表 | 收入分配越平等 |
| 不计入当年GDP的 | 二手车交易、股票购买、财政转移支付 |
| 年租金属于什么收入 | **财产性收入** |
